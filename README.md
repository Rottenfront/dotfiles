# Rottenfront's Dotfiles
My own dotfiles collections for *NIX systems

## Linux configuration
There are configs for 2 window managers - DWM and Hyprland.
Default applications packages:
- `neovide` - GUI for NeoVim
- `microsoft-edge-dev` - browser
- `blueberry` - bluetooth connection
- `visual-studio-code-bin` - VSCode

### Hyprland
Required packages:
- `hyprland`
- `rofi-wayland` - application launcher
- `waybar` - topbar
- `swaybg` - wallpapers
- `foot` - terminal
- `grimshot` - screenshots

### DWM
Right now DWM config is very incomplete. Please do not use it.
DWM and st binaries should be compiled from config.
Required packages:
- `xorg`
- `feh` - wallpapers
- `scrot` - screenshots
